\section{Projects}

\begin{twenty}
	\twentyitem{---}{[2018-2019] Building a simple-tensor: Simplification of tensorflow operations}{}{
		This project is creating an opensource python package for simplifying the usage of tensorflow. It contains tensor operation, loss, and common architecture modules. Our package also includes some computer vision algorithm modules, ex:
		
		1. Object detection(YOLO-V3) module (from scratch): With the "simple-tensor" you can modify the size of your YOLO-V3 model up to 30 times smaller or 6.7 MB than the original (>200MB).
		
		2. Image classification module: A simplification of the tensorflow image classification. It depends on tensorflow slim as base architectures. Currently, "simple-tensor" only support for resnet, densenet, and inception-v4.
		
		3. Segmentation module: Deepl-lab
		
		Github: \href{https://github.com/fatchur/Simple-Tensor} {Github Link}
		
		Pypi: \href{https://pypi.org/project/simple-tensor/} {How to install}
	}
	
	\twentyitem{Qoala.id}{[2019] Computer Vision for Automated Claim}{}{We are working with some algorithms like object detection, segmentation, classification, keypoint detection, face alignment, and recognition for building a system of automating insurance claim by videos or images. 	\twentyitem{Side Job}{PT. Pertamina, Unit Balongan Refinery, Vision Based Safety Analysis.}{}{We developed a vision-based video analysis for Balongan Refinery workers. The system can detect violations, ex: not using helm or standard jacket.  
		
		This project depends on some technologies:
		
		1. Opencv: Streaming the video from CCTV, manipulating the image.
		
		2. Kafka: Streaming the data between containers
		
		3. Redis: Saving the temporary data, ex: camera URL
		
		4. Tensorflow: Inferencing the input image
		
		5. Gunicorn, Gevent, Flask: Webserver
